# Capstone-Project-Modul-1

Capstone Project dengan study case berupa data karyawan perusahaan. Tujuan dari project ini adalah untuk memperlihatkan hasil pembelajaran modul 1 yaitu python fundamental.
Adapun materi yang diaplikasikan kedalam project ini adalah data collection, conditional statement, looping, regular function,
dan beberapa materi lainnya. Pengaplikasian materi tersebut dituangkan kedalam sebuah program sederhana yang terdiri dari beberapa fitur utama, yaitu create, read, update, dan delete.
modul yang digunakan pada project ini adalah datetime, dateutil, prettytable, dan termcolor.
